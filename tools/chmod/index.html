Unix 文件权限计算器：勾选 r/w/x 自动生成数字（如 755）与符号（如 rwxr-xr-x）格式，支持 setuid/setgid/sticky 特殊位。
特殊位
读 (r)
写 (w)
执行 (x)
所有者 (Owner)
用户组 (Group)
其他人 (Other)
数字格式 755
符号格式 rwxr-xr-x
命令 chmod 755 file
常见权限参考
数字符号说明
000----------无权限
400r--------仅所有者可读
444r--r--r--所有人可读
600rw-------仅所有者可读写（私钥文件）
644rw-r--r--常规文件默认
664rw-rw-r--共享文件
666rw-rw-rw-所有人可读写
700rwx------仅所有者可读写执行（~/.ssh）
755rwxr-xr-x可执行文件/目录默认
777rwxrwxrwx所有人可读写执行（危险）
4755rwsr-xr-xsetuid（如 passwd 命令）
2755rwxr-sr-xsetgid（共享目录）
1755rwxr-xr-tsticky（如 /tmp）
