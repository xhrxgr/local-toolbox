AES 加解密，使用浏览器原生 Web Crypto API，支持 AES-GCM（含认证，推荐）与 AES-CBC。密钥可由密码经 PBKDF2 派生（10 万次迭代，SHA-256），或直接输入 hex 密钥。全部本地运行，密钥不离开浏览器。

古典密码，支持凯撒 / ROT13 / ROT47 / 维吉尼亚 / Atbash / XOR / 仿射。ROT13、ROT47、Atbash、XOR 为对称算法，加密解密按钮等效。
编码工具（非加密），支持 Base64 / Base32 / Hex / 摩斯电码 / 二进制 / URL / HTML 实体。Base64 与 Hex 使用 UTF-8 安全处理。
